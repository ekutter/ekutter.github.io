MTP Sync tool

Aimed at garmin devices.  The purpose is to aid in copying files to/from Media Transfer Protocol devices. 

Use at your own risk: I'm not going to guarantee you won't lose a file.  Bugs happen.  But I haven't run into any issues in several months.

Any questions or issues please email me at eric@mtyquinn.com

usage: download the following file: MTPSync.zip. You can place MTPSync.exe anywhere on your computer and just create a convenient link to it.  It will default to using C:\MTPSync for the cache location.  Through the menu's you can change that to anywhere. That is where the configuration file lives.  If you change the folder, you'll lose all your current settings.

My personal settings for each device are: auto-sync, assign cache drive letter, auto delete activities

Features

  • Keep Track of your devices
  • Works with MTP and non MTP devices alike
  • Auto detect - looks for MTP devices every 5 seconds.  Performs sync on connected devices.  enable/disable in File menu.
  • Show connection status
  • show last sync time
  • Allow assigning drive letter for quick access to cache folder for a specific device.  Really handy for the CIQ sdk - m:\apps for the target. Note, it will create a network drive with the link. You probably have to be an administrator. Force an update of the connections with "File\Update Network Drives"
  • Automatically deletes files after sync (optional for activities and scorecards)
  • Aliases names: different devices use different folder names for activities for example: Activity vs Activities.  Always uses Activity on PC.
  • trim old files in cache (you got to tell it with File\Trim) - it'll get really big otherwise
  • rename files with date/device name on import when appropriate
  • hide devices you don't often use
  • Quick link to folders - double click on model name or drive letter to bring up folder in explorer.  No way to automatically bring up device folder on MTP.
  • File menu items generally are global
  • Device menu items are generally device specific settings/operations
  • Logging - all copy and delete operations are logged to MTPSyncLog.txt located in the same folder as MTPSync.exe. use "File / View Log File" to view in notepad.  Feel free to delete it if it's getting too big.

- MTP syncs the following folders from your device to a folder on you computer:

  • Activities
  • Scorecards
  • scrnshot
  • Logs\Dbg, Logs\Bin
  • Apps\Logs - and append .bak to .txt files in the process.  Truncates .txt files after sync.
  • Apps -> Syncs PC to device. pre-build to convenient drive letter - auto copy to device when connected
  • GarminDevices.xml - saves one per day / device
