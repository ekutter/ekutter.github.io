MTP Sync tool

Aimed at garmin devices.  The purpose is to aid in copying files to/from Media Transfer Protocol devices. 

Use at your own risk: I'm not going to guarantee you won't lose a file.  Bugs happen.  But I haven't run into any issues in several months.

Any questions or issues please email me at eric@mtyquinn.com

usage: download the following file: MTPSync.zip. You can place MTPSync.exe anywhere on your computer and just create a convenient link to it.  It will default to using C:\MTP for the cache location.  Through the menu's you can change that to anywhere. That is where the configuration file lives.  If you change the folder, you'll lose all your current settings.

Features

  • Keep Track of your devices
  • Show connection status
  • show last sync time
  • Allow assigning drive letter for quick access to cache folder for a specific device.  Really handy for the CIQ sdk - m:\apps for the target
  • Automatically deletes files after sync (optional for activities and scorecards)
  • Aliases names: different devices use different folder names for activities for example: Activity vs Activities.
  • trim old files in cache (you got to tell it with File\Trim) - it'll get really big otherwise
  • rename files with date/device name on import when appropriate
  • hide devices you don't often use

- MTP syncs the following folders from your device to a folder on you computer:

  • Activities
  • Scorecards
  • scrnshot
  • Logs\Dbg, Logs\Bin
  • Apps\Logs - and append .bak to .txt files in the process
  • Apps -> pre-build to convenient drive letter - copy to device when connected
  • GarminDevices.xml
