Dog Tracker App
See your dogs location with a glance at your wrist when paired with a Garmin Alpha or Astro 430/320.

Features
  • Customize-able distance alerts - set the distance, enter tone, and exit tone
  • Optional repeating distance vibration - vibrates every 10 seconds if your dog stays outside of a custom set distance
  • Optional movement alert - short vibration when they stop, longer one when they start moving again.
  • dog odometer - displays the distance your dog has traveled since you started the app. Can be disabled
  • Single display to show direction of multiple dogs and distance to first 4
  • Individual display for each dog
  • Configurable Alpha ID for multiple Alpha/Astro environment.
  • Indicates if dogs are moving or stationary
  • Indicates North on the summary page and, on some devices, with a small red circle around the edge of individual dog pages
  • Displays distance in yards or meters, depending on user settings
  • Multiple Garmin devices can be displaying data from a single Alpha at the same time as long as they are within a few yards of the Alpha.
  • use next/prev (page up, page down, screen swipes) to move between dogs.
  • low Alpha battery symbol will show next to the current time if battery is low
  • low collar battery symbol will show next to the dog distance if the collar battery is low
  • The app will display "searching" until it finds an Alpha.
Usage
  • Please make sure you have the latest firmware for your Alpha/Astro 430 and "settings / Dogs / Broadcast Dog Data" set to On.
  • The app will display "no dogs" if there are no dogs configured or connected to the Alpha.
  • The data being displayed is from a connection to the Alpha hand held. This does not work with just the dog collar.
  • On the individual dog pages, the arrow turns green when the dog is moving, red when the dog is stopped.
  • On the group dog page (big circle), there will be a small direction indicator for each dog, using the color configured on the Alpha itself, and the distance of the furthest dog. The name of the furthest dog will be displayed above the distance.
  • On the group dog page, the big circle will turn green if the furthest dog is moving and red if she is stationary.
  • On the group dog page, the outline of each of the direction indicators will be thick if the dog is stationary, thin if the dog is moving.
  • The direction arrow is relative to the direction your watch/device is pointing. For best results your watch/device should be held level in front of you.
Configuring Settings
  • In Garmin Connect on your phone, select your "watch device / Connect IQ Apps / Applications / Dog Tracker App / settings"
  • For the Alpha ID in settings, set it to "-1" to lock in the next Alpha found. Use "0" to have it always use the first Alpha it finds, but without locking it in for next time. Or enter the specific Alpha ID you want to always find. You can determine the Alpha ID by setting it to "-1" when no other Alpha's are around. It will then fill in this setting with the current Alpha ID.
  • Alerts - Set the distance and tones. If you change one of the enter or exit tones (a number from 0 - 19) and and press save while your watch is connected to your phone, it will play the tone on update. This lets you try out various tones. A value of 0 means no tone.
  • distance alert of zero turns off the distance alerts
Trouble Shooting
  • Please make sure you have the latest firmware for your Alpha/Astro 430 and "settings / Dogs / Broadcast Dog Data" set to On.
  • Troubles connecting to your Alpha: it may be due to conflicts with Garmin's Dog Tracker Widget. To fix it:
    • Run the Garmin Widget (down arrow from the watch face until you see it)
    • Press the Start button to bring up the menu
    • Select “forget this device”
    • Exit out of the Widget with the Back button
  • "no dogs" : You have a connection to the Alpha/Astro but it doesn't see any dogs. This is a connection issue between your Alpha/Astro and the dog collar, not with the Dog Tracker App.
