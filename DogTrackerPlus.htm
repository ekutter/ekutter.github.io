Features
  • Compatible with most Alpha and Astro devices
  • Map with dog tracks / location + detailed map on compatible watches (limit 1 on detailed maps)
  • Dog distance alerts - customizable tones
  • Dog moving/stopped alerts
  • Dog odometer

  • Movement indication for each dog
  • Battery status for tracker and dog collars
  • Distance displayed in yards or meters, depending on user settings
  • Upto 20 dogs - depending on your Tracker device
Usage
  • Start page - press "menu" to configure, or "start" to begin tracking once your Tracker is connected and GPS is found
  • Press start button again when you are done with your activity.
  • A Tracker device (Alpha or Astro) is required.
  • Dog colors are configured on the Tracker device itself - note colors don't always 100% match on the watch
  • On map pages, you can use the "back" button to zoom on the map.
  • Map pages: circles indicate the dog is moveing, squares indicated they are stopped
  • Arrows turns green when the dog is moving, red when stopped.
  • Multi dog ring - Red/Green shows movement status of furthest dog
  • Multi dog ring - For each dog, direction symbol has bold outline when dog is stopped
  • Direction arrows are relative to the direction your watch/device is pointing. For best results, hold your watch level in front of you.
  • Detailed maps - for best results, before launching Dog Tracker Plus, configure the watch to show "North Up".
  • If GPS or the Tracker lose signal, a small red "GPS" or "Dogs' will show towards the top

Configuring
All settings for Dog Tracker Plus are now done through menu's on the watch. You no longer need to use your phone. Press the "menu" button on the watch to bring up the menu.
Main Menu
  • Tracker Menu
    • Search for new Tracker
    • Lock current Tracker id for future sessions
    • Tracker id - use for this session
    • *Demo Mode - virtual tracker for demonstrations
  • Settings Menu
    • GeoFence - alert when your dog leaves/enters radius in yards/meters
    • GeoFence continuous - alert every 10 seconds while dog outside of radius
    • Exit Tone - tone when your dog leaves the GeoFence radius
    • Enter Tone - tone when your dog enters the GeoFence radius
    • Movement Alert - alert when your dog starts moving
    • Stop Alert - alert when your dog stops
    • *Record Activity - record activity and get distance, pace, time, etc.
    • *enable GPS - required ofr activity recording
    • *Clear All Settings - resets all settings (display, alerts, etc)
  • Display Options
    • Configure Page - fields to be displayed on each data page
      • Set Header - set the top field for the page
      • Set Field - set the main fields for the page
    • Show Activity Labels - show labels below each field. For bigger text, turn this off.
    • Draw Alert Circle - draw a circle showing the dog alert radius.
    • Show Dog Ring - always shown with more than 2 dogs. If turned off with one or two dogs, shows the dog dist/dir on either side of the screen.
    • *Map Refresh Rate - interval to update dog location and tracks on the detailed map page.  Longer interval reduces flicker
  • Diagnostics
    • Send Diagnostics now - Contact Developer
    • Show Diagnostics
    • Automatic Diagnostics - with this enabled, the app will occasionaly send annoymous diagnostics information to the developer so that Dog Tracker Plus can be improved based on how it is being used. This information has no user identifying information and will only be used for Dog Tracker Plus improvement. It will not be shared with any third parties, including Garmin, and will not be used for profit in any way.
*setting may not be available while tracking or on some devices
Trouble Shooting
  • Please make sure you have the latest firmware for your Alpha/Astro and "settings / Dogs / Broadcast Dog Data" set to On.
  • Troubles connecting to your Alpha: it may be due to conflicts with Garmin's Dog Tracker Widget. To fix it:
    • Run the Garmin Widget (down arrow from the watch face until you see it)
    • Press the Start button to bring up the menu
    • Select “forget this device”
    • Exit out of the Widget with the Back button
  • Tracker still won't connect - you may need to reboot your Tracker device or your watch (power off/on). Some devices have more issues with this than others.
  • Tracker Connected, but no dogs are found - This is a connection issue between your Alpha/Astro and the dog collar, not with the Dog Tracker App. Make sure you can see the dogs on the hand held.
  • menu doesn't work on detailed map page - currently a limitation of the devices
  • detailed slow to update and map flashes - currently a limitation of the devices
  • Detailed maps only can show one dog track due to device limitations
  • Settings all have to be reset from earlier versions.
  • There are no labels on the dog list data page.  This page will always show the bigger text.
